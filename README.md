按鍵 TAB + 按鍵 Q 同時按 = ESC   
按鍵 P + 按鍵 - 同時按 = DEL   
按鍵 win_code + 按鍵 win_num 同時按 = Layer windown_function_layer   
按鍵 mac_code + 按鍵 mac_num 同時按 = Layer mac_function_layer   
To 4 = 切換至 mac_default_layer   
To 8 = 切換至 game_default_layer   
   
## td_multi_win   
第一下或按住 = WIN_GUI   
快速點兩下 = 打開截圖   
快速點三下 = 打開 Windows 終端機
